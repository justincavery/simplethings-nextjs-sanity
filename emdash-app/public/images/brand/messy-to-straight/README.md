# Messy to Straight Variations

These files explore the Simple Things idea of taking messy technical systems, workflows, or delivery paths and making the important route clearer.

- 01-thin-brand-line-over-black-mess: Thin black disorder with a direct brand-colour path from the same start to end.
- 02-heavy-brand-line-over-black-mess: Same metaphor with more weight, better for favicon-sized viewing.
- 03-black-only: No colour, just contrast between noisy path and clean path.
- 04-bordered: A contained, company-logo treatment with a clear outer frame.
- 05-unbordered-open: The lightest mark, useful if the crop or surrounding UI already frames it.
- 06-dotted-mess-solid-line: The messy line becomes audit-trail dots; the answer is continuous.
- 07-dotted-solution: A lighter, strategic-planning feel where the straight line is a dotted route.
- 08-harsh-change: A hard before-and-after: chaotic left side, clean right side.
- 09-gradual-radiance: A softer change with glow, useful for the AI/process-improvement angle.
- 10-reverse-direction: Right-to-left version, useful to test how direction changes the reading.
- 11-diagonal-direction: Upward direction: more growth/momentum, less literal process cleanup.
- 12-black-background-field: Dense black/grey messy field in the background with one confident brand line over it.
- 13-thin-black-field-red-line: Thin black messy lines everywhere, with one red line from the same start to end.
- 14-thin-black-field-dark-orange-line: The same field idea with a darker orange solution line.
- 15-red-background-cut-through: A brand-colour block where the clean line cuts through the mess in cream.
- 16-linework-only: No dots, no frame, no fill tricks: just the before path and the better path.
- 17-hairline-random-red: Much thinner black random mess with a red straight line from the same start to end.
- 18-hairline-random-dark-orange: A darker-orange answer line over a more random hairline black field.
- 19-sparse-random-no-dots: A quieter random field with no endpoints, keeping the clean line as the only strong object.
- 20-dense-hairline-field: Denser and more chaotic, but still using thin black background strokes.
- 21-faint-chaos-bold-answer: Very faint thin black disorder with a stronger straight brand line.
- 22-black-ground-hairlines: Black ground with thin warm-grey chaos and one dark-orange answer line.
- 23-single-wide-red: One continuous black line takes a wide messy route, with a red straight line from the same start to end.
- 24-single-tall-orange: One smooth black path wanders vertically before the dark-orange direct route resolves it.
- 25-single-compact-red: A tighter single-line tangle for better small-size recognition.
- 26-single-edge-orange: One messy black path uses more of the icon boundary, then resolves to a direct dark-orange line.
- 27-single-low-red: One lower, looser black path with a red straight line cutting across it.
- 28-single-black-only: Only one messy black line, with no coloured solution line.
- 29-single-sparse-red: A simpler single black path with fewer turns and a red direct line.
- 30-single-black-ground: A single warm-grey messy line on black, with one dark-orange direct line.
- 31-single-spaghetti-red: One true-black smooth path crosses and loops more randomly, with a red direct line.
- 32-single-crossing-orange: One true-black smooth path moves more randomly around the space, resolved by dark orange.
