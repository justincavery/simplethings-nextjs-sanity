A few years ago I was introduced to NFTs and I fell in love.

Stupid, I know. The heart wants what it wants.

My first NFT was the profile image I still carry today, and onchain SVG Avastar that I bought from OpenSea for a few hundred dollars because it was bearded, had messy hair, and Australia green and gold colours.

The second NFT I bought was Hidenki's Singularity project on Art Blocks (and later traded for a slightly better one on the NFTX vault swap).

It was from that community that I found out about the Anomoslous Machines and the Machine for Dying, a nifty little SVG animated set of tiled NFTs.

&lt;Longer story to come&gt;

I realised that they pulled down their website hosting the images because it was too expensive to run the infrastructure they had going on at AWS. I asked for the code to get the data so that I could try setup something to keep the beautiful artwork going... which resulted in [https://diabolicalmachines.simplethin.gs](https://diabolicalmachines.simplethin.gs)
